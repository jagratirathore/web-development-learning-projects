// Adding Information in URLs (Query Strings):

// hmari APIs (jo URLs/endpoints hote h) me hm additional information
// query string ke through bhej skte h

// example:
// jb hm browser me www.google.com likhte h to ek request server ko jati h
// ab agar hm search me "apple" likhte h to URL change ho jata h:

// https://www.google.com/search?q=apple

// yha "?" ke baad jo part aata h use query string bolte h

// q = key (query)
// apple = value

// yani hm key-value pair ke form me additional information bhej rhe h

// hm direct URL me bhi likh skte h:
// https://www.google.com/search?q=mango
