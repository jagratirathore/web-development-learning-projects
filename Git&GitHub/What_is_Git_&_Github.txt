What is Git?

Git is a free and open-source Version Control System.
(It is a tool that helps to track changes in code.)

👉 Simple words me:
Git ek Version Control System hai jo code me hone wale changes ko track karta hai.

🔹 Detailed Explanation (Easy Hinglish)

Jab hum kisi project par kaam karte hain (especially company ya team me), to project ek hi din me complete nahi hota — usme multiple stages hoti hain.

Example:

Pehle HTML likhte hain
Phir CSS add karte hain
Phir JavaScript likhte hain

Ya phir har feature (jaise Signup / Login) me bhi stages hoti hain:

Step 1 → HTML
Step 2 → CSS
Step 3 → JavaScript
⚠️ Problem Without Git

Maan lo:

Tumne Step1 → Step2 → Step3 complete kar liya
Lekin Step3 (JS) me error aa gaya

Ab agar tumhe Step1 ya Step2 wali previous code version chahiye, to:

Ya to manually code delete karna padega ❌
Ya guess karna padega kya change kiya tha ❌
✅ Solution: Git

Git kya karta hai:

Har change ko track karta hai
Project ki history save karta hai
Tum kisi bhi previous version pe wapas ja sakte ho
👨‍💻 Team Work Example

Company me:

Project par multiple developers kaam karte hain
D1, D2, D3, D4

Git help karta hai:

Kisne kya code likha
Kis file me changes kiye
Kaunsa code final rakha jaye
Sab log ek hi project par easily collaborate kar sakte hain
⭐ Features of Git
Track history of code
Collaboration (team work easy banata hai)
Free & Open Source
Fast & Scalable
Most popular tool (almost sab companies use karti hain)

👉 Git ko tum resume me “Tools & Technologies” section me add kar sakte ho ✔️

✅ What is GitHub?

GitHub is a website where we host Git repositories online.
(GitHub is based on Git.)

🔹 Simple Hinglish Explanation

GitHub ek online platform/website hai jahan:

Hum apne projects (repositories) upload karte hain
Apna account/profile banate hain
📱 Real-Life Example

Jaise:

Instagram par log apni photos dalte hain
Waise hi developers GitHub par apne projects upload karte hain
💼 Job Perspective

Jab tum job ke liye apply karte ho:

Resume ke saath GitHub profile link bhi bhejte ho

Hiring manager check karta hai:

Tumne kaunse projects banaye
Kaunsi technologies use ki
Tumhari coding skills

👉 Isse tumhari authenticity aur chances dono increase hote hain ✔️

📁 Important Term
Repository (Repo) = Project ka folder (jisme code hota hai)
🔥 Final Difference (Short)
Git	GitHub
Tool hai	Website hai
Code track karta hai	Code host karta hai
Offline bhi use hota hai	Internet required
Version control system	Git-based platform
