🔹 Basic Commands

init → New Git repository create karne ke liye

git init

Add Remote Repository

git remote add origin <repository-link>

Check Remote

git remote -v

Check Branch

git branch

Rename Branch to main

git branch -M main

Push to GitHub

git push origin main
🔹 When We Create Project Locally (Without GitHub)

👉 Kai baar hum project direct GitHub par nahi banate,
balki apne local system (laptop) par banate hain.

Example:

Humne delta-demo folder ke andar ek folder banaya:

project2

Check karne ke liye:

ls

👉 Sirf project2 folder dikhai dega

Uske andar jaane ke liye:

cd project2

Ab agar:

ls

👉 To folder empty hoga (koi file nahi)

🔹 Check Hidden Files
ls -la

👉 Output:

.
..

📌 Matlab:

Abhi tak ye folder Git repository nahi hai
Git is project ko track nahi kar raha

🔹 Convert Folder into Git Repository

👉 Ab hum project2 ko Git repository banana chahte hain

Command:
git init

👉 Output:

Initialized empty Git repository
🔹 After git init

Phir se check karo:

ls -la

👉 Ab output me dikhega:

.
..
.git ✅
🔹 Important Point

👉 .git folder ka matlab:

Ab ye folder Git repository ban chuka hai
Git ab is project ke changes ko track karega
🔹 Now You Can Do

Ab tum is project me:

git add
git commit
git push

sab use kar sakte ho ✅
