🔹 Basic Git Commands

clone → Repository ko local machine par copy karna

git clone <repository-link>

status → Code ki current state (changes, modified files, etc.) dikhata hai

git status
🔹 What is Clone?

👉 Clone ka matlab: copy ya duplicate banana

Jab hum GitHub par koi repository bana lete hain, wo online (GitHub) par hoti hai, lekin hamare local system (laptop) me nahi hoti.
Agar hum us repository ko apne system me lana chahte hain, to hum clone karte hain.

✔️ Matlab:

Online repo waha rahegi hi
Aur uski ek copy hum apne system me le aate hain

📌 Example:
Jaise Instagram se photo download karke apne system me lana → same concept = clone

🔹 Steps to Clone a Repository
GitHub par apni repository open karo
Code button par click karo
3 options milte hain:
HTTPS
SSH
GitHub CLI

👉 Beginners ke liye HTTPS best aur simple hota hai (recommended)

HTTPS link copy karo
VS Code me apna folder (e.g., GITDEMO) open karo

Terminal me command likho:

git clone <paste-link-here>

✔️ Cloning ke baad ek new folder (e.g., delta-demo) create ho jaayega

🔹 Check Files After Cloning
ls
Isse tumhe cloned files dikh jaayengi
Jo file GitHub par thi (e.g., README.md), wo ab local system me bhi hogi
🔹 Navigate Inside Repository
cd delta-demo
ls -a
ls -a → hidden files bhi show karta hai
Yahan tumhe .git folder milega

👉 .git folder important hota hai

Ye sirf Git repositories me hota hai
Ye proof hai ki folder Git-controlled hai
🔹 Why We Use Clone?

👉 Clone ka main reason:

Project me changes / updates karna
Local system par kaam karke phir GitHub par update karna
🔹 Making Changes in Files
VS Code me README.md file open karo
Niche content add karo:
# Teacher
Shradha Khapra

# Student
Delta Student

👉 # use karne se text bold/heading ban jata hai (Markdown)

Changes save karo (Ctrl + S)
🔹 File Status After Changes
VS Code me README.md ke aage M (Modified) dikhai dega
File ka color white se yellow ho jaayega

👉 Iska matlab:

File me changes ho chuke hain
Lekin abhi Git me save (commit) nahi hua

📌 Important:
Git me sirf file change karna enough nahi hota
👉 add aur commit bhi karna padta hai

delta-demo folder ke andar new files create ki:
index.html → ! + Enter se boilerplate code
style.css

👉 Git ka use karte time hum terminal me commands type karte hain
👉 Aur Git files normal files se different hoti hain (tracking + version control)
