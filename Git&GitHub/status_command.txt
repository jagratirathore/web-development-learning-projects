🔹 Status Command in Git

git status command ka use hum code ki current state dekhne ke liye karte hain.

👉 Jab bhi hum files me changes karte hain, unka status change hota rehta hai
Aur ye command hume batata hai ki file kis state me hai.

✅ Command:
git status
🔹 Real-Life Example (Letter)

Socho tumne ek letter likha:

Letter likha → normal state
Envelope me daala + stamp lagaya → status change
Post kar diya → final

👉 Matlab letter ki ek life cycle hoti hai

📌 Isi tarah Git me files ki bhi ek life cycle (states) hoti hai

🔹 Git File Lifecycle
Untracked → Unmodified → Modified → Staged → Unmodified (after commit)

Ya detail me:

Untracked --(git add)--> Staged  
Staged --(git commit)--> Unmodified  
Unmodified --(edit file)--> Modified  
Modified --(git add)--> Staged  
Staged --(commit)--> Unmodified  
Unmodified --(remove file)--> Untracked
🔹 Understanding Each State
1️⃣ Untracked (New Files)
Git ko pata hi nahi hota ki ye file exist karti hai
Git is file ko track nahi kar raha hota

👉 Agar tum is file ko Git ka part banana chahte ho:

git add filename
2️⃣ Unmodified (Tracked but No Changes)
File Git ke control me hai
Lekin abhi usme koi changes nahi hue

👉 Ye normal state hoti hai (no changes)

3️⃣ Modified
Jab tum file me changes (edit/update) karte ho
Tab file modified state me aa jaati hai
4️⃣ Staged
Jab tum git add karte ho
Tab file staging area me aa jaati hai

👉 Matlab:

Ye file commit ke liye ready hai
5️⃣ Commit (Back to Unmodified)
Jab tum commit karte ho:
git commit -m "message"

👉 Tab:

Changes permanently save ho jaate hain
File fir se unmodified state me aa jaati hai
🔹 Important Concept
Git har chhote se chhote change ko track karta hai
Lekin sirf un files ka jo tracked (add ki hui) hoti hain
Untracked files ki history Git ke paas nahi hoti
🔹 Check Status Example
git status

👉 Output me tum dekhoge:

README.md → Modified (kyunki tumne usme changes kiye the)
New files → Untracked
🔹 Summary
git status → current state batata hai
File lifecycle hoti hai (Untracked → Modified → Staged → Commit)
Git me changes save karne ke liye:
👉 add + commit zaroori hai
