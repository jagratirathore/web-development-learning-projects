// Terminal se related kaafi terms hoti hain jo kabhi-kabhi interchangeably use hoti hain,
// lekin technically sabka meaning thoda different hota hai (same nahi hote).

// Terminal: Ek interface jisme hum commands type karte hain.

// Different Terms:

// Command Line:
// Aisa interface jahan user textual commands enter karke system se interact karta hai.
// (Generally Windows me use hota hai - Command Prompt)

// Terminal:
// Ek program jo command line interface provide karta hai.
// (Mac/Linux me commonly use hota hai)

// Console:
// Ek command-line interface jo direct computer system ke sath interact karta hai.

// Shell:
// Ek program jo terminal ke andar run hota hai aur commands ko execute karta hai.

// Bash (Bourne Again Shell):
// Ek popular shell jo Linux aur Mac me widely use hota hai.

// Z-Shell (zsh):
// Bash ka advanced version, Mac me by default pre-installed hota hai.

// Note:
// Pehle ke time me command line, terminal aur console hardware devices hote the.
// Example: ATM machine ko terminal samajh sakte hain,
// aur uske andar jo program run hota hai wo shell hota hai.

// Shell ke types:
// - Bash Shell (Linux/Mac me popular, Windows me Git Bash ke through use hota hai)
// - Z-Shell (Mac me pre-installed)

// Windows users ke liye:
// Saari commands easily run karne ke liye Git Bash install karna useful hota hai.
